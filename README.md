# PerlFuck Transpiler

## Overview

The PerlFuck Transpiler is a simple Perl script that translates a string of code into a series of Perl expressions. It is designed to be an educational project that pushes the limits of the language.

## Prerequisites

Before using the PerlFuck Transpiler, make sure you have the following prerequisites installed on your system:

- Perl: The script is written in Perl, so you need a Perl interpreter to run it. Most Unix-like systems come with Perl pre-installed. You can also download and install Perl from the [official Perl website](https://www.perl.org/).

## Usage

There are 2 ways to use PerlFuck! You can either go to the [web converter](https://antoinepoulin.com/perlfuck) or follow the steps below for the CLI tool:

To use the PerlFuck Transpiler in the CLI, follow these steps:

1. Clone or download the repository to your local machine.
2. Open a terminal or command prompt and navigate to the directory containing the `perlfuck.pl` script.
3. Run the script with the code you want to translate as a command-line argument:

   ```bash
   perl perlfuck.pl 'code as a string'
   ```
4. The script will output the Perl code translated in PerlFuck to the terminal

## Example

Here's an example of how to use the PerlFuck Transpiler:
```bash
perl perlfuck.pl 'print \"Hello World!\";'
```
